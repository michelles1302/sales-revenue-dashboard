# 📈 Sales & Revenue Analysis Dashboard

## Overview

This project is an interactive Sales & Revenue Dashboard built using Streamlit, Pandas, and Plotly.

The dashboard allows users to upload sales data in CSV or Excel format and analyze business performance through KPIs, charts, filters, and business insights.

---

## Features

* Upload CSV and Excel files
* Interactive dashboard using Streamlit
* KPI tracking

  * Total Revenue
  * Total Orders
  * Average Order Value
  * Quantity Sold
* Revenue trend analysis
* Top-performing products analysis
* Region-wise sales distribution
* Category-wise revenue analysis
* Business insights generation
* Download filtered reports in CSV and Excel formats

---
